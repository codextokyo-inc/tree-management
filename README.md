# re:leaf - 樹木管理システム

🌳 森林や樹木を効率的に管理するためのWebベースの地図アプリケーション

## 📖 概要

re:leafは、樹木の位置情報、管理情報、売却意思、病気の状態などを地図上で視覚的に管理できるシステムです。
地形情報や地質情報と組み合わせて、効率的な森林管理・搬出計画を立てることができます。

## ✨ 主な機能

### 🗺️ 地図機能
- **Leaflet.js** ベースのインタラクティブな地図
- **OpenStreetMap** による詳細な基盤地図
- ドラッグ可能なマーカーでポイント管理
- ポリゴン描画によるエリア管理

### 🌳 ポイント管理
- **樹木ポイント**: 樹種、サイズ、樹齢、剪定履歴、病気情報、売却意思、搬出難易度
- **事業者ポイント**: 造園、販売、輸出入、管理者、行政、森林組合、製材所
- 住所検索によるポイント追加（国土地理院API対応）
- カテゴリ別フィルタリング

### 📊 集計・分析
- ポリゴン範囲内のポイント自動集計
- 売却意思別、病気有無別、搬出難易度別の統計
- CSV/JSON形式での一括エクスポート・インポート

### 🗺️ 地形・環境レイヤー
- **陰影起伏図**: 地形の起伏を視覚化
- **地質図**: 産総研シームレス地質図（凡例付き）
- **色別標高図**: 標高を6段階で色分け（凡例付き）
- **浸水深レイヤー**: 国土地理院の洪水想定区域（凡例付き）

### 📏 測定・ルート設定
- **距離測定**: 2点間の直線距離を測定
- **ルート設定**: 複数地点を結ぶルート計画
- 測定結果・ルートの保存と管理

## 🚀 クイックスタート

### 前提条件
- Python 3.x がインストールされていること
- モダンなWebブラウザ（Chrome, Firefox, Edge など）

### ローカルでの起動

#### Windows
```cmd
start_server.bat
```

#### Mac / Linux
```bash
python3 -m http.server 8000
```

その後、ブラウザで以下にアクセス:
```
http://localhost:8000/map/
```

## 🛠️ 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **地図ライブラリ**: Leaflet.js, Leaflet.draw
- **データ保存**: localStorage API

## 📊 データ管理

### エクスポート形式
- **JSON形式**: 完全なデータ構造を保持
- **CSV形式**: ZIP圧縮、ポイント/ポリゴン/測定線/ルート別

### ストレージ
- ブラウザの localStorage に自動保存
- 手動エクスポートによるバックアップ推奨

## 🔧 主な機能の使い方

### ポイント追加
1. 左側パネル「操作」→「📍 ポイント追加」
2. カテゴリを選択（樹木 or 事業者）
3. 地図上をクリックして配置
4. 詳細情報を入力して保存

### 住所検索でポイント追加
1. 「🏠 住所から追加」をクリック
2. 都道府県から正確に住所を入力
3. ポイントタイプを選択
4. 「🔍 検索して追加」をクリック

### エリア集計
1. 「📐 エリア描画」をクリック
2. 地図上でポリゴンを描画
3. 自動的に範囲内のポイントを集計

### 距離測定
1. 「📏 距離測定」をクリック
2. 測定したい2点のポイントを順にクリック
3. 直線距離が表示される
